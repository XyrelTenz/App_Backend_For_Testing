# 🚖 BaoBao (EasyBao) Backend

The **BaoBao Driving App** backend is a modern, high-performance Kotlin and Spring Boot 3 application. Migrated from a Go-based architecture, it provides a scalable and secure foundation for real-time ride-hailing services.

## Overview

Designed for production-ready deployment, the system manages the complete lifecycle of ride-hailing interactions—from secure user registration (Firebase & Native Auth) to intelligent geospatial ride matching (PostGIS) and real-time communication (WebSockets & FCM).

The codebase is fully containerized and includes a comprehensive automated test suite to ensure reliability across all core modules, including authentication, passenger services, driver management, and real-time notifications.
